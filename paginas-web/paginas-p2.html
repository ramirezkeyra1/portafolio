Tarea 2: Maquetación de una página web

En esta actividad copié y pegué los códigos de mis archivos HTML y CSS en un documento en blanco y realicé capturas de pantalla de la maquetación en el navegador para comprobar su funcionamiento. La página está estructurada con un encabezado dividido en tres secciones, un menú de navegación, un Main con tres apartados y un pie de página. Esta práctica me permitió comprender cómo organizar una página web de manera ordenada.

Tarea 3: Propiedades de CSS para el Menú de navegación

En esta actividad revisé un archivo de Classroom sobre las propiedades de CSS e identifiqué cuáles permiten mejorar el estilo del menú de navegación. Además, copié mis códigos HTML y CSS en un documento y realicé capturas de pantalla de mi página para comprobar el diseño aplicado. Aprendí a utilizar CSS para optimizar la apariencia y la organización del menú de navegación dentro de una página web.
